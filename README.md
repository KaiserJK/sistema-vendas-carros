# Sistema de Vendas de Carros

Projeto acadêmico em Python utilizando arquivos JSON para cadastro de carros, clientes, funcionários e registro de vendas.

## Funcionalidades

- Cadastro de carros
- Cadastro de clientes
- Cadastro de funcionários
- Registro de vendas

## Tecnologias

- Python
- JSON

## Como executar

```bash
python src/Vendas.py
